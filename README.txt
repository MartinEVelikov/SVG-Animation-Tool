За стартиране на проекта Ви е необходим XAMPP!
1. Стартирате XAMPP - Кликвате върху Start бутона за Apache и за MySQL.
2. В phpMyAdmin качвате базата от данни (скрипта createDb.sql в папката config).
3. В браузър, през localhost, отваряте проекта и започвате.
